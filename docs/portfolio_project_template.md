# SCAN.LAB: Шаблон проекта для портфолио (бриф + материалы)

Дата: 2026-02-17<br>
Назначение: заполнить по каждому проекту, чтобы быстро собрать карточку в `portfolio.html` и подробный кейс (страницу проекта).

Рекомендуемый процесс:

1. Скопируйте этот файл для каждого проекта (например, `input/portfolio/<slug>.md`).
2. Заполните поля и вставьте ссылки на материалы.
3. Если данных нет или нельзя публиковать, укажите `NDA`/`нет данных`.

---

## 0) Требования к ссылкам на изображения (Google)

Ссылки должны открываться **без авторизации** и отдавать картинку напрямую (не страницу Drive с кнопкой “Скачать”).

Подходы, которые обычно работают:

- `lh3.googleusercontent.com/...` (прямая картинка)
- Google Drive файл:
  - share URL вида `https://drive.google.com/file/d/<FILE_ID>/view?...`
  - direct download URL вида `https://drive.google.com/uc?export=download&id=<FILE_ID>`

Если direct‑link требует подтверждение/не скачивается (часто бывает на больших файлах), приложите альтернативный direct‑link (или уменьшенную версию файла).

Для каждого изображения указывайте:

- `url` (прямая ссылка)
- `alt` (короткий alt‑текст)
- `credit` (если нужен источник/автор)
