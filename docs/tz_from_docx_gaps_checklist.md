## 1) Что в проекте уже покрыто

- Структура лендинга/сайта и ключевые страницы: есть главная, услуги, процесс, портфолио, кейс, контакты, FAQ.

- Преимущества (мобильность/автономность/масштаб/помощь с ТЗ): реализовано на главной (`index.html`) и в `advantages-faq.html`.

## 2) Что покрыто частично

- **Интеграции (формы/карта/аналитика) как конфиг**:
  - Форма: реализован success/error flow + stub-режим, endpoint задается через `<meta name="scanlab:form-endpoint" ...>`.
  - Карта: iframe embed включается через `<meta name="scanlab:map-embed-url" ...>`, иначе показывается placeholder.
  - Аналитика: есть `assets/analytics.js` и `meta`-конфиг для Я.Метрики/GA4, включается только при наличии ID (и не на localhost/file).
