- Преимущества (мобильность/автономность/масштаб/помощь с ТЗ): реализовано на главной (`index.html`) и в `advantages-faq.html`.

## 2) Что покрыто частично

- **Хедер по ТЗ.docx**:
  - требуется: логотип + контакты (Telegram/телефон/почта) + CTA “Написать в Telegram”.
  - сейчас: в хедере есть логотип/навигация/CTA “Рассчитать стоимость”, но контактов и Telegram-CTA в хедере нет.
- **Блок “О компании”**:
  - требуется: “с 2015 года”, направления, сферы, фото офиса/оборудования.
  - сейчас: на главной есть блок преимуществ, но нет отдельного “О компании” с фактами/фото.
- **Портфолио как набор кейсов 5–6 шт с полями**:
  - требуется: фото + описание + сроки + (опц.) “до/после” по кейсам.
  - сейчас: карточки есть, but сроки/состав материалов/структура “что собрать по кейсу” не унифицированы (часть карточек без сроков/клиента).
- **Контакты: карта**:
  - требуется: карта (опционально).
  - сейчас: блок карты в `contacts.html` пока декоративный (без embed).
- **Футер**:
  - требуется: копирайт + ссылки на политику + оферту.
  - сейчас: есть копирайт + `privacy.html`, но нет страницы/ссылки “Оферта / пользовательское соглашение”.
